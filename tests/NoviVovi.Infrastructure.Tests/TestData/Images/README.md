# Тестовые изображения для экспорта в Ren'Py

## 📁 Структура папки

Положите сюда тестовые изображения для мануальных тестов экспорта:

```
Images/
├── background.png      # Фон (рекомендуемый размер: 1920x1080)
├── character.png       # Персонаж (рекомендуемый размер: 500x1000)
└── README.md          # Этот файл
```

## 🖼️ Требования к изображениям

### background.png (Фон)
- **Формат**: PNG
- **Рекомендуемый размер**: 1920x1080 (Full HD)
- **Описание**: Любое изображение фона (комната, улица, пейзаж)
- **Пример**: Можно использовать любую картинку интерьера или пейзажа

### character.png (Персонаж)
- **Формат**: PNG
- **Рекомендуемый размер**: 500x1000 (портрет в полный рост)
- **Описание**: Изображение персонажа с прозрачным фоном
- **Пример**: Аниме-персонаж, силуэт человека, или любой другой персонаж

## 🎨 Где взять тестовые изображения

### Вариант 1: Создать простые заглушки
Можно создать простые цветные прямоугольники с текстом в любом графическом редакторе.

### Вариант 2: Использовать бесплатные ресурсы
- **Фоны**: [Unsplash](https://unsplash.com/), [Pexels](https://www.pexels.com/)
- **Персонажи**: [OpenGameArt](https://opengameart.org/), [Itch.io](https://itch.io/game-assets/free)

### Вариант 3: Сгенерировать программно
Тесты автоматически создадут fake PNG изображения, если реальные файлы не найдены.

## 🧪 Использование в тестах

Тесты `ManualExportTest` и `ManualExportLargeNovelTest` будут:
1. Проверять наличие файлов в этой папке
2. Если файлы найдены - использовать их
3. Если файлов нет - создавать fake PNG (простые заглушки)

## ⚠️ Важно

- Файлы в этой папке **НЕ коммитятся** в git (добавлены в .gitignore)
- Каждый разработчик может использовать свои тестовые изображения
- Размеры изображений могут быть любыми, но рекомендуется придерживаться указанных
